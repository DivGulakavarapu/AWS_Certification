Amazon CloudWatch Events
It is a web service that monitors your AWS resources and the applications you run on AWS. 
You can use Amazon CloudWatch Events to detect and react to changes in the state of a pipeline, stage, or action. 

CloudWatch Metric
When you create an alarm you specify 3 settings
1. Period
2. Evaluation Period
3. Datapoints to Alarm
